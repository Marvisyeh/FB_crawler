# FB crawler (fb 公開社團爬蟲)
<!-- - FB crawler is an crawler for Facebook group, written in python. -->
- 爬取社團中的所有貼文
    - 貼文時間
    - 貼文編號
    - 貼文者名稱
    - 留言數量
    - 貼文網址
    - 貼文內容
    - 留言內容

- 輸出 JSON 格式
> 每一千則貼文輸出一個檔案
